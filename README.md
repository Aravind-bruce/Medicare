# MediCare – Premium Medical Landing Page

Pixel-perfect React landing page built with Vite, Tailwind CSS, and Framer Motion.

## Features
- Fully responsive (Mobile, Tablet, Desktop)
- Dark/Light mode toggle
- Smooth animations with Framer Motion
- Reusable components and clean folder structure
- Semantic HTML and accessibility

## Tech Stack
- React 18, Vite, Tailwind CSS, Framer Motion, Lucide React

## Installation
```bash
npm install
npm run dev
